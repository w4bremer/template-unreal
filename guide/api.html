# API

As in every programming tutorial we use a Hello World example and create the following API. However, in our example we are doing a bit more than just saying hello to the world.

HelloWorld Cube Counter API (click to expand) 
schema: apigear.module/1.0
name: Hello.World
version: "1.0"

interfaces:
  - name: CounterCube
    properties:
      - { name: color, type: Color }
      - { name: size, type: Size }
      - { name: count, type: int }
    operations:
      - name: increment
        params:
          - { name: step, type: int }
        return:
          type: int
      - name: decrement
        params:
          - { name: step, type: int }
        return:
          type: int
      - name: randomize
      - name: reset
    signals:
      - name: orientation
        params:
          - { name: orientation, type: Orientation }
enums:
  - name: Size
    members:
      - { name: Small, value: 0 }
      - { name: Medium, value: 1 }
      - { name: Large, value: 2 }
structs:
  - name: Color
    fields:
      - { name: red, type: int }
      - { name: green, type: int }
      - { name: blue,  type: int }
  - name: Orientation
    fields:
      - { name: x, type: float }
      - { name: y, type: float }
      - { name: z,  type: float }
  - name: ColoredOrientation
    fields:
      - { name: color, type: Color }
      - { name: orientation, type: Orientation }

# Interface

We have a cube which is also able to count. Therefore we call the interface CounterCube. Aside from the count property, the cube also has a color and a size. The operations increment and decrement could be use to change the count property by a certain step size. randomize could set all properties to a random value and reset restore the default values. The signal orientation could contain the orientation of the cube in the 3D space.

interfaces:
  - name: CounterCube
    properties:
      - { name: color, type: Color }
      - { name: size, type: Size }
      - { name: count, type: int }
    operations:
      - name: increment
        params:
          - { name: step, type: int }
        return:
          type: int
      - name: decrement
        params:
          - { name: step, type: int }
        return:
          type: int
      - name: randomize
      - name: reset
    signals:
      - name: orientation
        params:
          - { name: orientation, type: Orientation }
